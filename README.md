# AI Resume Filtering System

An NLP-based resume filtering system that analyzes uploaded PDF resumes and recommends suitable job roles using Sentence Transformers and semantic similarity.

## Features

- PDF Resume Upload
- Skill Extraction
- Buzzword Detection
- Competency Analysis
- Skill Ageing Analysis
- Job Role Recommendation
- Semantic Matching using MiniLM Embeddings

## Tech Stack

- Python
- Pandas
- Sentence Transformers
- Scikit-learn
- Streamlit

## How it Works

1. Upload Resume PDF
2. Extract Text
3. Generate Embeddings
4. Compute Cosine Similarity
5. Extract Skills and Buzzwords
6. Recommend Suitable Job Roles


## Model

MiniLM-based Sentence Transformer embeddings with Cosine Similarity for semantic job-role matching.
